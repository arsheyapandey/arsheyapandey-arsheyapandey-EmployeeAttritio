# Employee Attrition
This project developed an Employee Attrition Prediction system using the IBM HR Analytics dataset. The data was preprocessed by removing irrelevant features, encoding categorical variables using One-Hot Encoding, converting the target variable (Attrition) to binary format, and scaling numerical features using StandardScaler. The dataset was split into 80% training and 20% testing sets, and class imbalance was addressed using class_weight='balanced' for applicable models.

Three classification algorithms were trained and evaluated: Logistic Regression, Random Forest Classifier, and Gradient Boosting Classifier. Performance was measured using Precision, Recall, F1-Score, and ROC-AUC Score. Logistic Regression achieved the best overall performance with a Precision of 0.356, Recall of 0.660, F1-Score of 0.463, and the highest ROC-AUC Score of 0.804, making it the most effective model for this imbalanced dataset. Although Random Forest and Gradient Boosting achieved higher precision, they had significantly lower recall, resulting in lower F1-scores.

Exploratory Data Analysis revealed that the Sales department experienced the highest attrition rate, with Sales Representatives and Laboratory Technicians showing the highest employee turnover. The analysis also indicated that employees with lower monthly income, frequent overtime, lower work-life balance, and fewer years at the company were more likely to leave. Feature importance analysis from the Logistic Regression model further confirmed that variables such as OverTime, MonthlyIncome, YearsAtCompany, Age, and JobSatisfaction were among the strongest predictors of employee attrition. These findings can support HR teams in identifying high-risk employees and implementing proactive retention strategies.
